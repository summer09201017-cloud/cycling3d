# 3D 自行車競速(cycling3d)

> HFPC 3D 系列(2026-07-20,fork 自 speedskating3d)——奧運/公路自行車橢圓賽道
> (兩直道+兩個 180° 彎道,柏油路面),核心手感=**左右交替節奏踩踏**(左右腳交替踩踏板)
> +**彎道傾身**(自行車過彎本來就要傾身)。夏季戶外皮;含同機雙人(duel-2p-kit §7C 競速型)。

## 玩法

- **單人競速**:跟 AI 車手內外道對決,先騎完全部圈數(難度定 2~3 圈)的贏。
- **雙人同機**:P1(紅)=A/D 交替踩+W 傾身;P2(藍)=←/→ 交替踩+↑ 傾身,內外道各一人。
- **練習場**:無對手、無限圈,自由練踩踏節奏。

左右**交替**按鍵=踩踏板(左腳/右腳):交替且節奏穩=越騎越快(踩一下衝一下);連按同側或太急
=踉蹌掉速(溫柔,不摔倒、永遠騎得完)。進彎道 HUD 會提示——**按住傾身鍵**內傾過彎不減速,
沒按=彎道自然減速(不懲罰性)。單人模式方向鍵是 P1 的別名(沒有死鍵);平板點畫面=自動左右交替踩。

- 五難度:幼兒(強輔助+AI 慢+2 圈)→ 職業(踏頻窗窄+AI 快+3 圈)。速度上限比滑冰高(自行車較快)。
- P1 紅衣、P2 藍衣、AI 綠衣;騎士戴安全帽+車衣、踩踏動畫、自行車(車架/兩輪/握把/曲柄踏板);
  臉部鐵則(眼耳嘴眉)照 3d-figure-kit。

## 開發

```bash
npm install
npm run dev                      # 本機試玩(dev 不註冊 SW);run.bat=port 5220
npm run build                    # 產物在 dist/
node scripts/gen-voice.mjs       # 烤人聲(雲哲 14 句;產物進 git,離線可玩)
node scripts/verify-cycling.mjs <url> <outDir>  # Playwright 端到端(單人×2難度/雙人/練習/彎道傾身)
```

引擎重點:解析式 stadium 賽道(`ovalPoint(dist, laneOffset)` 里程域,直道+180° 彎,
分道=法線偏移)、節奏判定 `q = 1 - |gap - ideal|/tol`、racer 結構 P1/P2/AI 統一只差輸入來源、
`this.running` 只給主迴圈 RAF(fork 血緣沿用的撞名事故鐵則)。踩踏動畫:每次有效踩踏 `strideT += 0.5`
(半圈動力衝程)+連續遞增,曲柄/輪子隨之轉(`poseCyclist`)。
人聲鐵律:預烤 mp3(雲哲),缺檔只出字幕、絕不用 Web Speech 機器聲。
